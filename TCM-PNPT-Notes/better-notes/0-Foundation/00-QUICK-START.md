# PENTESTING QUICK START GUIDE

## Five Stages of Ethical Hacking
1. **Reconnaissance** - Gather info (passive/active)
2. **Scanning** - Identify hosts, ports, services
3. **Enumeration** - Extract detailed service info
4. **Exploitation** - Gain access
5. **Privilege Escalation** - Get root/admin

---

## Workflow Checklist
- [ ] Passive recon (OSINT)
- [ ] Active recon (scanning)
- [ ] Enumerate services
- [ ] Research vulnerabilities
- [ ] Craft exploits
- [ ] Post-exploitation/privesc

---

## Common Ports Quick Reference
| Port | Service | Notes |
|------|---------|-------|
| 22 | SSH | Remote access |
| 80 | HTTP | Web |
| 443 | HTTPS | Secure web |
| 445 | SMB | Windows shares |
| 3306 | MySQL | Database |
| 3389 | RDP | Remote desktop |
| 8080 | Alt HTTP | Proxy, apps |

---

## Key Tools by Phase
- **Recon:** Hunter.io, IntelX, Sublist3r, WHOIS
- **Scanning:** Nmap, Nessus, Masscan
- **Enumeration:** Nikto, DirBuster, SMB client
- **Exploitation:** Metasploit, SearchSploit, Custom scripts
- **Privesc:** LinPEAS, WinPEAS, GTFOBins

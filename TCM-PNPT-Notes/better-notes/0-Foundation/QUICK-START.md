# PENTESTING QUICK START GUIDE
## Five Stages
1. **Reconnaissance** - OSINT/passive recon
2. **Scanning** - Identify hosts/ports
3. **Enumeration** - Service details
4. **Exploitation** - Gain access
5. **Privilege Escalation** - Root/admin
## Common Ports
| Port | Service |
|------|---------|
| 22 | SSH |
| 80 | HTTP |
| 443 | HTTPS |
| 445 | SMB |
| 3306 | MySQL |
| 3389 | RDP |
| 8080 | HTTP Alt |
## Key Tools
- **Recon:** Hunter.io, IntelX, Sublist3r
- **Scanning:** Nmap, Nessus, Masscan
- **Enumeration:** DirBuster, FFUF, impacket
- **Exploitation:** Metasploit, SearchSploit
- **Privesc:** LinPEAS, WinPEAS, GTFOBins
- **Post:** Responder, ntlmrelayx, hashcat

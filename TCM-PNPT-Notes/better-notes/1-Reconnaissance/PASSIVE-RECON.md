## Commands
```bash
# Website technology identification
$ whatweb example.com

# With custom user agent (spoofing)
$ whatweb -a 3 --user-agent "Mozilla/5.0 (Windows NT 10.0; Win64; x64)" https://example.com

# DNS info
$ nslookup example.com
$ dig example.com
```
